# 🤖 Ahmed-Mini-Bot v1.0

**Simple, Lightweight WhatsApp Bot**

## ✨ Features

- ✅ `.ping` - Check response time
- ✅ `.alive` - Show bot status
- ✅ `.menu` - Display commands
- ✅ Beautiful QR code pairing
- ✅ Works on Railway (FREE)
- ✅ 24/7 always online

## 🚀 Deploy in 5 Minutes

### Step 1: GitHub
```
1. github.com/new
2. Name: Ahmed-Mini-Bot
3. Make PUBLIC
4. Upload all files
```

### Step 2: Railway
```
1. railway.app
2. New Project
3. Deploy from GitHub
4. Select Ahmed-Mini-Bot
5. Wait for GREEN ✅
```

### Step 3: Add Number
```
Railway → Variables:
OWNER_NUMBER = +923001234567
```

### Step 4: Connect WhatsApp
```
1. Open: https://your-app.railway.app/pair
2. Scan QR code
3. Enter your number
4. Click "Connect Bot"
5. Done! ✅
```

### Step 5: Test
```
Send to bot:
.ping → Response time
.alive → Bot info
.menu → All commands
```

## 📱 Commands

```
.ping       Check bot response time
.alive      Show bot information
.menu       Display this menu
```

## 🔧 Configuration

Edit `.env` file:
```
OWNER_NUMBER=+923001234567
BOT_NAME=Ahmed-Mini
PREFIX=.
MODE=public
```
